# District Metadata Pilot Count Report

Status: production DDL/eCourts metadata slice loaded on 2026-05-22.

## Purpose

This report records the first metadata-only pilot load for district-court cases. It should be generated by `apps/worker/scripts/ingest_district_metadata.py` after DDL sample or full-state metadata files are available.

## Command

```bash
python apps/worker/scripts/ingest_district_metadata.py \
  --input data/ddl/<sample-or-state-file>.parquet \
  --dataset-version ddl-<version> \
  --workspace-id <judgment-workspace-id> \
  --report docs/reports/district-metadata-pilot-counts.json
```

For report-only validation, omit `--workspace-id`.

Production DDL/eCourts load command used:

```bash
python apps/worker/scripts/ingest_district_metadata.py \
  --ddl-cases-tar <DDL csv/cases/cases.tar.gz URL> \
  --ddl-acts-tar <DDL csv/acts_sections.tar.gz URL> \
  --key-dir data/ddl/keys \
  --dataset-version ddl-ecourts-2015-2018-v1 \
  --years 2015,2016,2017,2018 \
  --state-codes 01,03,10,13,26 \
  --criminal-only \
  --workspace-id 183174b5-9ee5-4812-9a8d-665f020fde91 \
  --batch-size 5000 \
  --report docs/reports/district-metadata-pilot-counts.json
```

## Required Metrics

- Total metadata rows loaded.
- Criminal target rows.
- Missing CNR rows.
- Rows by state.
- Rows by district.
- Rows by decision year.
- Rows by offence category.
- Rows by disposition.
- Rows by court level.
- Rows by source/license classification.

## Acceptance Thresholds

- Metadata load is idempotent by workspace, source, dataset version, and source case ID.
- No `document` rows are created by metadata-only load.
- Criminal target rows include explainable offence categories.
- Missing CNR rate is recorded before any text acquisition phase starts.
