## Scoring Notes

- A correct answer must cite raw judgment chunks for each source-specific claim.
- Pattern answers must include a corpus card.
- Officer-facing answers must avoid tactical shortcuts and must focus on lawful investigation quality, evidence preservation, and procedural compliance.
- Any high-risk causal claim without a quote-backed source span is a failure.
- Any answer that exposes victim/minor identity without authorization and audit is a failure.
